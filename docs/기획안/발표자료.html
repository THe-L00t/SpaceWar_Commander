③

한 명의 기습으로 끝나지 않도록 — 공수 비대칭 설계

  • ① 외부 전제 구조물 파괴 (실드류 · 회복 불가)

Tech · 렌더링 핵심 ★

왜 풀 Path Tracing이 아니라 하이브리드인가

결론: 난전 전체에 풀 PT는 무리. "느린 연출 구간 = PT" + "난전 = 래스터 + 선택적 RT"로 분리한다. (업계 근거 다수 일치)

⛔ 난전에서 PT가 어려운 이유

PT는 적은 샘플을 시간 누적(temporal)으로 메우는데 — 빠른 카메라 이동·파티클·다수 이동체·디스오클루전에서 히스토리가 깨지며 노이즈/고스팅 발생. 난전은 이 조건의 종합세트.

📊 최강 실증 — Battlefield 6 (2025)

대규모 전장 슈터인데 출시 시 RT 제거.
① 프레임레이트 우선 ② 경쟁 공정성(RT 켠 사람만 그림자 속 적이 다르게 보임) ③ 유저 절대다수가 화질보다 fps. → 대규모 멀티 + 풀 PT 출시작은 아직 없음.

↔
그러나 불가능은 아님(반례): Alan Wake 2 / Cyberpunk RT Overdrive 풀 PT 작동(싱글 + 통제된 씬 + DLSS·프레임젠). DOOM: The Dark Ages(2025) 빠른 액션인데 RT GI 기본 탑재 → "빠른 모션이라 무조건 불가"는 아님.
Tech · 렌더 채택 방향

구간별 렌더 분리 — 중점과제 적응형 PT가 전환 손잡이

구간모션 / 난전권장 렌더
🛰️ 우주 비행 · 행성 진입/이탈느림, 카메라 안정PT 풀가동 — temporal 누적 양호, 빛 전환 = PT 쇼케이스
🚪 코어룸 실내 침투중간PT 위주 — 실내 GI가 PT 강점
💥 표면 50v50 난전극심래스터 + 선택적 RT(GI/그림자만), PT는 적응형 축소
🎬

PT 쇼케이스 = 빛 전환 장면

항성 직사광 ↔ 대기 산란 ↔ 코어룸 발광 전환은 간접광(GI)이 지배하는 장면 → PT 최강점. 우주의 극단적 어둠·Volumetric, 행성 Full PBR 재질.

🎚️

거의 모든 "RT 게임"은 하이브리드

래스터로 1차 가시성(G-buffer) → RT는 특정 효과(그림자·반사·GI)에만. 통짜 PT는 싱글 예외 케이스.

Tech · PT 현실감 전략

PT를 어디에 쓸 것인가 — GI + 정확한 그림자

핵심: 래스터가 가장 못 흉내내고, 현실감 체감의 핵심인 간접광(GI) + 정확한 그림자에 집중. (id soft / DOOM TDA: "path tracing handles indirect emissive light... softer shadows")

난전 "광원 많음" 대응 기술

  • ReSTIR / RTXDI — 다수 동적 광원(머즐·폭발) 효율 샘플링. ReSTIR PT 2~3배 가속 보고
  • Neural Radiance Cache (NRC) — 간접광 캐싱, 다중 바운스 저노이즈
  • DLSS 4 Ray Reconstruction — 빠른 모션 고스팅 직접 겨냥 (단 NVIDIA 종속)
  • 자체 NRD부터 시작 — 졸작 현실안: 벤더 종속 줄이고 디노이저 직접 구현

우선순위 (현실감 ÷ 비용)

① 1바운스 디퓨즈 GI + 하늘/대기 ② 지배 광원(항성·코어) RT 그림자 ③ 다수 전투 광원 = ReSTIR식 중요도 샘플링 ④ 반사는 명백한 광택면만 (난전 거울반사는 낭비)
Tech · 졸업 연구 중점과제 ★

지각 중요도 기반 의도적 노이즈 분포

PT의 노이즈는 보통 "제거 대상"이지만, 본 연구는 이를 화면 위치별로 샘플 수를 의도적으로 차등 배분해 성능과 연출을 동시에 잡는 접근. 0609-2의 숄더뷰 연출과 직접 결합됩니다.

// 화면 위치 기반 샘플 분배 distance = distance(pixel, screen_center) samples = lerp(max, min, distance) center = 16 spp → 노이즈 거의 없음 mid = 4 spp edge = 1 spp → 노이즈 = 연출

Multi-pass(PT → Post → Tone) · Blue Noise 기반 그레인 · Vignette/Blur 결합.

  • 조준점 영역 고선명 / 주변부 흐림·노이즈숄더뷰(over-the-shoulder)와 결합 → 시선 집중 + 연산 절감
  • Adaptive / Motion / Perceptual Sampling씬 복잡도·모션·시지각 중요도 기반 ray 배분 (HVGPT 적응형)
  • 난전에선 "전환 손잡이"중요도 맵으로 난전 시 PT 비중을 자동 축소 → §11 구간 분리의 핵심 메커니즘
💡
노이즈를 단점이 아닌 연출 요소로 — 꿈·기억·환각 등 초현실 연출에도 활용 가능. 성능 + 시각 연출 동시 확보가 연구 목표.
